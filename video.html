01

Receiving a product

A new chemical arrives. Scan the supplier barcode and the system handles the rest.

Scan the barcode
Any supplier barcode, any package size
→
AI identifies it
Substance name, quantity, and package size confirmed automatically
→
Print a lab label
Add your name, date, storage location, expiry, and any other notes via the touchscreen
→
Apply and place
You choose where it goes. The system logs it.
02

Using and reordering

When a container is empty, one scan is enough to update stock and trigger an order if needed.

Log in and scan
Use your university account, then scan the label on the empty container
→
Stock check
System checks if the item is still available elsewhere in the lab before ordering
→
Order placed
If stock is gone, one confirmation places the order automatically
→
Email confirmation
Both the student and the lab technician get an email with order details and delivery time
03

Authorization for restricted items

Students cannot order hazardous or expensive chemicals directly. They submit a request and the technician decides.

Request submitted
Student submits a request through the system for a restricted chemical
→
Technician reviews
Lab technician approves or rejects, and adds a reason if rejecting
→
Student notified
Email with the outcome and, if rejected, the reason why
